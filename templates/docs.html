POST /api/graph/execute

Execute the LangGraph multi-agent orchestration system

Request Body: 

{
  "company_name": "TechCorp Inc",
  "industry": "Software & Technology",
  "location": "San Francisco, CA",
  "total_budget": 100000,
  "target_days": 90,
  "objectives": [
    "Launch SaaS platform",
    "Build enterprise sales team"
  ],
  "thread_id": "optional-uuid",
  "use_checkpointing": true
}

Parameters:

company_name string required

Company name for the project

industry string required

Business industry

location string required

Company location

total_budget number required

Total project budget ($1,000 - $1,000,000)

target_days number required

Project timeline in days (30-365)

objectives array required

List of strategic objectives

thread_id string optional

Resume previous execution by thread ID

Response:

{
  "success": true,
  "thread_id": "uuid-here",
  "message": "Execution started"
}

WebSocket Events:

execution_started

Emitted when graph execution begins

agent_update

Agent status changes (idle/active/completed/error)

phase_update

Execution phase transitions with progress percentage

execution_complete

Final results when all agents complete
